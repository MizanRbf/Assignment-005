Question-1: What are the different ways to select an element in the DOM?

We can select an element in various ways, such as getElementById(), getElementsByClassName(), getElementByTagName(), querySelector(), querySelectorAll() etc.

Question-3: What is event delegation in the DOM?

In javaScript It works more and more. We can control child element by parent element with event delegation. That is to say, we don't need to attach evenListener to all child element. We will attach evenListener to parent element and work easily.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a important concept. Which is used in DOM.It is also closely related concept with event delegation. When a work occurs in a element it is also occurs in the child element.

Question-5: How do you create, add, and remove elements using JavaScript?

We can do dynamic works by javaScripts. We can create a new element by using document.createElement(). We can also add this element after creating by using appendChild() as well as we can remove the previous element or new element by using removeChild().
